# pytask-master
A personal project that monitors windows processes against a list of "time wasting processes" in programs.txt.
It tells you via voice command that you're wasting time and you should stay on task.

# Requirements:
<ul>
  <li>win32pdh </li>
  <li>string </li>
  <li>win32api </li>
  <li>pyttsx3</li>
</ul>
